# 支持 Intel macOS 13+

## 目标

- 生成 **仅 Intel(x64)** 的 macOS 安装包（DMG/ZIP）。
- 运行时依赖检测与安装脚本支持 **macOS 13+**。

## 计划步骤

1. **构建配置支持 x64**

- 更新 [`package.json`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/package.json) 的 `build.mac.target[].arch`，加入 `x64` 或改为仅 `x64`（根据 Intel-only 需求）。
- 在 [`ninja/build.sh`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/ninja/build.sh) 增加 `--x64` 参数处理，配套新增 `build:x64` 脚本。
- 同步 [`ninja/README.md`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/ninja/README.md) 的命令说明。

2. **macOS 版本门槛调整为 13+**

- 运行时检测：更新 [`src/scripts/ntfs-manager/dependencies.ts`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/src/scripts/ntfs-manager/dependencies.ts) 的版本检查逻辑与注释（>=13）。
- 安装脚本：更新 [`ninja/install-dependencies.sh`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/ninja/install-dependencies.sh)、[`ninja/kunai.sh`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/ninja/kunai.sh) 的提示文案与判断逻辑（>=13）。
- UI 文案：更新 [`src/locales/zh-CN.json`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/src/locales/zh-CN.json) 等多语言 `macosVersion` 文案为 13+（保持一致性）。

3. **说明与入口脚本文案一致化**

- 更新 [`docs/00-发布说明.md`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/docs/00-发布说明.md)、[`README.md`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/README.md)（若含 Apple Silicon/14+ 描述）为 Intel + 13+。
- 如需要，修正文案注释（如 [`ninja/nigate.sh`](/Users/zhangjiawei/Downloads/Free-NTFS-for-Mac-main/ninja/nigate.sh) 的“14+”注释）。

- macos-13: 调整依赖检测与安装脚本的版本门槛
